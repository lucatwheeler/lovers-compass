## Description

Love Compass is an intimate location-sharing app built for couples. Pair with your partner using a simple code, and a beautiful compass will always point in their direction, no matter how far apart you are.

**How it works:**
- One partner creates a couple and shares the pairing code
- The other partner enters the code to connect
- A live map shows both of your locations
- A compass heart always points toward your partner
- See the distance between you at a glance

**Features:**
- Real-time location sharing on a clean, beautiful map
- Compass indicator that always points to your partner
- Distance display showing how far apart you are
- Poke your partner to let them know you are thinking of them
- Background location updates so your partner can always find you
- Secure pairing with unique 8-character codes
- Privacy controls: pause sharing anytime
- No accounts, no email, no sign-up required

**Privacy first:**
- Only the latest location is stored -- no history, ever
- No analytics or tracking
- No third-party data sharing
- Unpair to permanently delete all data
- Device IDs are random, not linked to your identity

Love Compass is designed for two people. It is not a social network. It is not a tracker. It is a simple, beautiful way for couples to feel connected, wherever they are.

## App Review Notes
Love Compass is a private location-sharing app for couples. To test:
1. Install on two devices (or one device and one simulator)
2. On Device A, tap "Create Couple" to generate a pairing code
3. On Device B, enter the code and tap "Join Couple"
4. Both devices will now see each other's location on the map
5. The compass indicator points toward the partner's location
6. The "Poke" button sends a notification to the partner

The backend is hosted at https://web-production-558a2.up.railway.app -- you can verify it is live by visiting /health in a browser.

Location permission is required for core functionality. Background location allows the partner to see updates when the app is not in the foreground.
